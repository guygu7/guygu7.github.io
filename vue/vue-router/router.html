Vue-router 路由的简单配置与使用

首先说明 Vue-router的路由 是对组件进行渲染控制，并不会刷新页面。

在点击路由后会修改URL #号后的部分，例：

点击前：http://localhost:8080

点击后：http://localhost:8080/#/routes1  

关于 URL# 可以百度 URL hash

main.js

import Vue from 'vue'
import App from './App'
import routes1 from './components/Routes1.vue'
import routes2 from './components/Routes2.vue'

Vue.config.productionTip = false

// -------------路由---------------
import VueRouter from 'vue-router'
Vue.use(VueRouter)
 
//配置路由（需要将组件在这里配置给VueRouter才可作为路由使用）
const router = new VueRouter({
  routes: [
    { path: "/routes1", component: routes1 },
    { path: "/routes2", component: routes2 },
  ],
  //路由模式为历史模式
  //mode: "history"
})
// --------------------------------
 
new Vue({
  router,//路由
  el: '#app',
  render: h => h(App),
})

App.vue

<template>
  <div id="app">
    <router-link to="/routes1">路由1</router-link>
    <router-link to="/routes2">路由2</router-link>
    <router-view></router-view>
  </div>
</template>
<script>
export default {
  name: "App",
  data() {
    return {};
  },
  watch:{
    //对路由进行监听
    '$route.path':function (newVal,oldVal){
      window.console.info("新路由："+newVal+";旧路由："+oldVal);
    }
  }
};
</script>
