style

使用对应的sass-loader less-loader后可以直接在 .vue 文件中写 scss less

基于vue-cli的示例

Example.vue

<template>
  <div>
    style样式
  </div>
</template>
<script>
//关于 Vue 实例中的 this: 在Vue实例中，要获取 data 中的数据 或者 methods 中的方法，必须使用 this. 访问，这里的 this 表示 Vue 实例对象
export default {};
</script>
<style lang="scss" scoped>
/* scoped 表示仅当前组件有效，不加scoped则会全局生效，一般都加上 */
div{
  color:blueviolet;
}
/* 启用 scss less 需要设置 lang 属性 */
body{
  div{
    font-style: italic;
  }
}
</style>
