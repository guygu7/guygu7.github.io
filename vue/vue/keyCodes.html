keyCodes按键修饰符 按键捕获（按键监听）

需要注意：特殊按键F2等，在不同机器上可能值会不一样；

                VUE 1.x版本和2.x版本的别名定义方式的区别；

本示例演示了在input内获得焦点时，监听到按键，并获取到按键的值。

基于vue-cli的示例：

main.js

import Vue from 'vue'
import App from './App.vue'
//自定义全局按键修饰符
Vue.config.keyCodes.f2 = 174;
//1.x版本 Vue.directive('on').keyCodes.f2 = ?;
new Vue({
  render: h => h(App)
}).$mount('#app')

Example.vue

<template>
  <div>
    <div>
      keyup(可以捕获特殊按键):<input type="text" @keyup="getKeyCode($event)">
      keypress(部分特殊按键无法捕获):<input type="text" @keypress="getKeyCode($event)">
    </div>
    <!-- 内置: .enter .tab .delete(删除 和 退格) .esc .space .up .down .left .right
        可通过全局 Vue.config.keyCodes 自定义别名 -->
    <div>
      <!-- 特殊按键F2等，在不同机器上可能值会不一样不一定是174 -->
      F2:<input type="text" @keyup.174="f2press">
      <!-- 使用全局自定义别名修饰符 Vue.config.keyCodes.f2 = 174; -->
      F2全局自定义别名：<input type="text" @keyup.f2="f2press"><br>
    </div>
  </div>
</template>
<script>
//关于 Vue 实例中的 this: 在Vue实例中，要获取 data 中的数据 或者 methods 中的方法，必须使用 this. 访问，这里的 this 表示 Vue 实例对象
export default {
  methods:{// methods 属性中定义了当前 Vue 实例所有可用方法
    getKeyCode:function(ev){
      window.console.info(ev.keyCode);
      alert("捕获到当前按键keyCode为"+ev.keyCode);
    },
    f2press:function(){
      window.console.info("捕获到F2");
      alert("捕获到F2");
    },
  }
};
</script>
