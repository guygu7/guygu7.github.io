render函数

render是DOM渲染函数，因VUE推荐使用template，故本例只做简单说明。

基于vue-cli的示例：

Example.vue 

<template>
  <div>
    render后整个被替换,此处没有被替换
  </div>
</template>
<script>
//关于 Vue 实例中的 this: 在Vue实例中，要获取 data 中的数据 或者 methods 中的方法，必须使用 this. 访问，这里的 this 表示 Vue 实例对象
export default {
  //render是DOM渲染函数，VUE推荐使用template
  //vue 单文件里并没有 render 方法，因此这里不会执行
  render:function(creatElements){//render函数会将return中的组件完全替换本组件
    return creatElements('h',{props:{value:''}});
  },
  /**简化后的，main.js就是简化后的
   * render: h => h(Example9_Watch_Computed_)
   */
};
</script>
